12. Engenharia de tráfego: influenciar o upload manipulando o Weight do R8

R8: 
- ip prefix-list REDE20 permit 1.10.20.0/24
- route-map PIKACHU
- match ip address prefix-list REDE20
- set weight 666
- route-map PIKACHU permit 20
- router bgp 819
- neighbor 10.91.0.1 route-map PIKACHU in
- clear ip bgp * soft

- show ip bgp 1.10.20.0/24
- show ip route bgp

--------------------------------------------------------------------------------------------

13. Engenharia de tráfego: influenciar o upload manipulando o Local Preference do R8

R8:
- router bgp 819
- no neighbor 10.91.0.1 route-map PIKACHU in
- route-map BULBA
- match ip address prefix-list REDE20
- set local-preference 777
- route-map BULBA permit 20
- neighbor 78.0.0.2 route-map BULBA in
- clear ip bgp * soft

R9:
- show ip bgp 1.10.20.0/24

R10:
